# Update Block

<ApiEndpoint method="PATCH" path="/blocks/:id" />

Update an existing block. Only the attributes you include are changed.

## Request

::: code-group

```js [JavaScript]
import Confetti from 'confetti'

const confetti = new Confetti({ apiKey: 'your-key' })

const block = await confetti.blocks.update(210645, {
  type: 'text',
  status: 'published',
  slug: 'example',
  order: 0,
})
```

```sh [cURL]
curl -X PATCH "https://api.confetti.events/blocks/210645" \
  -H "Content-Type: application/json" \
  -H "Authorization: apikey your-key" \
  -d '{
  "data": {
    "type": "block",
    "id": "210645",
    "attributes": {
      "type": "text",
      "status": "published",
      "slug": "example",
      "order": 0
    }
  }
}'
```

:::
